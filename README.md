# Spys Bestelling Oplossing – Capstone Project
This project is a digital food ordering and delivery solution developed for Akademia’s on-campus catering service as part of our final-year Capstone (GIR) module.

Overview
The system allows students and staff to browse weekly menus, place meal orders in advance, and pay using a wallet-based system. Administrators and kitchen staff can manage menus, track orders, and monitor daily operations through a dedicated dashboard.

Features
Browse and pre-order meals

Wallet-based payment system

Role-based user access (students, staff, admin)

Order tracking and history

Admin tools for menu and order management

Technology Stack
Frontend: Flutter (Web & Mobile)

Backend: Supabase (Auth, Database, Storage, Edge Functions)
